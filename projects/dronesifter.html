Dronesifter

2020

A first step in 3D development. Currently an ongoing school assignment, this is the very first real project in 3D modeling and video game planning.



Maya

For a class assignment with an open prompt of creating a 3D interface, I chose a subject that would thrill me through my creation process. Minesweeper alone is a surpisingly fun logical puzzle game, and that is what makes it so ripe for creating alternative game mechanics to. My idea is a 3D version of the game so that the player is engulfed inside the environment. The aesthetic goal of the game is to use textures, shaders, and particle systems to make it truly a warm and almost magical environment. This could be considered for creating a virtual reality game as well.

Another part is a subtle story embedded in a little mascot: the drone. The robot's operative is to disarm all of the hidden bombs from within the landscape, or what could perhaps be considered to be a comet.
